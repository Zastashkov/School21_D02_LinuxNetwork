### 1.2. localhost

Для доступа к приложению, работающему на localhost ip адрес устройства должен входить в промежуток 127.0.0.1 — 127.255.255.254. 

- 194.34.23.100 - не localhost
- 127.0.0.2 - localhost
- 127.1.0.1 - localhost
- 128.0.0.1 - не localhost
